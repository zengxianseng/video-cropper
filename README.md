video-cropper 视频裁剪器

项目介绍

这是一个基于Python开发的带GUI界面的视频裁剪工具（终极版），采用tkinter搭建可视化界面，结合ffmpeg实现高效视频裁剪，支持图形化操作、进度显示、自动重命名保存，操作简洁直观，无需手动输入复杂命令，适合各类日常视频裁剪需求。

本项目为第一版，核心实现可视化视频裁剪功能，支持MP4格式视频，可自定义剪掉开头/结尾时长，实时显示裁剪进度，自动处理保存路径避免覆盖，后续可拓展更多视频格式支持及高级剪辑功能。

环境依赖

运行本工具需安装Python环境、相关Python依赖库，同时需确保系统中安装了ffmpeg（用于视频裁剪核心操作）：

1. 安装Python依赖库，执行以下命令：

pip install tkinter subprocess32

1. 安装ffmpeg：
            

  - Windows：下载ffmpeg安装包，配置系统环境变量（确保cmd中可直接运行ffmpeg命令）；

  - Mac/Linux：通过brew（Mac）、apt-get（Linux）直接安装：brew install ffmpeg 或 sudo apt-get install ffmpeg。

备注：tkinter通常随Python默认安装，若运行报错提示“no module named tkinter”，需手动安装对应依赖（Windows：pip install tk；Mac：brew install python-tk）。

项目结构

- Video_Cropping.py：项目主程序，包含tkinter GUI界面实现、ffmpeg裁剪逻辑、进度更新、异常处理等核心功能；

- Video_Cropping.spec：PyInstaller打包配置文件（用于将Python脚本生成可执行文件，方便无Python环境的电脑使用）；

- .gitignore：Git忽略文件，避免上传无关缓存、编译文件及系统临时文件。

注意事项

- 仅支持MP4格式视频，选择文件时请确认文件类型为MP4，否则无法正常加载；

- “剪掉开头”“剪掉结尾”需输入数字（可填小数），输入非数字会弹出错误提示，无法启动裁剪；

- 裁剪时长需合理：剪掉开头时长 + 剪掉结尾时长 需小于视频总时长，否则会提示“时间范围错误”；

- 必须安装ffmpeg并配置环境变量，否则会出现裁剪失败、报错等问题；

- 裁剪后的视频自动保存至原视频同目录，命名规则为“原文件名_cut.mp4”，若该文件已存在，将自动递增序号（如原文件名_cut_1.mp4），避免覆盖原文件；

- 裁剪过程中请勿关闭程序窗口，否则会导致裁剪中断，生成无效文件；

- 若运行报错，可优先检查：Python依赖是否安装完整、ffmpeg是否配置成功、视频文件路径是否正确。
