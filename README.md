video-cropper 视频裁剪器

编程语言

基于 Python 开发，结合 tkinter 搭建 GUI 界面、ffmpeg 实现视频裁剪核心功能。

核心功能

- 可视化操作界面，支持图形化选择视频文件；

- 可自定义剪掉视频开头、结尾时长（支持小数）；

- 实时显示裁剪进度，裁剪完成后提示保存路径；

- 自动处理保存路径，避免覆盖原文件；

- 提供打包好的 Windows 可执行文件（exe），无需 Python 环境即可直接运行。

注意事项

- 仅支持 MP4 格式视频，其他格式无法正常加载和裁剪；

- 裁剪时长需合理，剪掉开头与结尾的总时长需小于视频实际总时长；

- 运行 Python 脚本需安装 ffmpeg 并配置系统环境变量，运行 exe 文件无需额外配置；

- exe 文件仅支持 Windows 系统，且不可单独提取，需保留 dist 文件夹内相关依赖文件。

后续规划

本项目为第一版，后续将持续完善，拓展更多功能，包括但不限于支持更多视频格式、添加视频格式转换、水印添加等功能，优化界面体验。
